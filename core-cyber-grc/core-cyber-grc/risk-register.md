# Cyber Security Risk Register (Sample)

**Organisation:** Northbridge Health Services (fictional)  
**Sector:** Healthcare  
**Purpose:** Learning and demonstration of cyber security GRC principles

This risk register identifies key cyber security risks, assesses their impact and likelihood, and documents appropriate mitigation actions.

---

## Risk R-01: Excessive Access to Shared Folders

| Field | Description |
|------|------------|
| Risk Description | Sensitive information is stored in shared folders with excessive user access, increasing the risk of unauthorised access or data leakage. |
| Impact | High |
| Likelihood | Medium |
| Risk Rating | High |
| Risk Owner | Information Security Manager |
| Mitigation / Treatment | Review shared folder access permissions, apply role-based access controls, remove unnecessary access, and perform periodic access reviews. |
| Residual Risk | Medium |
